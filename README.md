Este projeto final de Programação I em Python consiste no desenvolvimento de um Sistema de Gerenciamento de Produtos que opera no terminal e implementa as quatro operações principais de um CRUD (Create, Read, Update, Delete). A estrutura de dados essencial do sistema é uma lista de dicionários, onde cada dicionário representa um produto com campos como ID, nome, preço e estoque. O funcionamento é baseado em um menu interativo, que guia o usuário pelas funcionalidades.
O menu principal apresenta seis opções claras para o usuário: 1. Cadastrar Produto (Create), que permite inserir novos registros de produtos, solicitando e validando as informações como nome, preço e estoque. 2. Listar Produtos (Read), que exibe de forma organizada todos os registros de produtos atualmente cadastrados na lista. 3. Atualizar Produto (Update), que permite a edição de um produto já existente após o usuário fornecer o ID do item a ser modificado. 4. Remover Produto (Delete), que exclui permanentemente um registro da lista, também requerendo o ID do produto para identificação. 5. Gerar Relatório/Resumo, que cumpre o requisito de apresentar uma pequena análise dos dados, como o total de produtos cadastrados, a quantidade total de estoque e o valor total do estoque (preço multiplicado pelo estoque). Por fim, a opção 6. Sair do Programa encerra o sistema de maneira segura e controlada. A organização do código segue o requisito de utilizar funções separadas para cada uma dessas operações.Por exemplo, para cadastrar um produto, o usuário digita '1', insere o nome (ex: "Teclado Mecânico"), o preço (ex: 250.00) e o estoque (ex: 15). Se o usuário escolher a opção para listar (opção '2'), o sistema imprime uma tabela com o ID, nome, preço e estoque de cada produto. Se a opção escolhida for atualizar (opção '3'), o sistema solicitará o ID do produto a ser editado e permitirá que o usuário altere seletivamente os campos. A funcionalidade de relatório (opção '5') fornece um resumo imediato do inventário, calculando o valor total do estoque e o estoque médio por produto.
